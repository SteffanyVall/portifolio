PROGRESSÃO ARITMÉTICA



Reflexão:

Progressão Aritmética é um conteúdo que é parecido com Progresão Geométrica, por esse motivo foi um dos conteúdos fáceis de se aprender e fazer também. Os dois tem relação entre eles e funções afins, assim ficando até melhor de aprender e fazer.


Sequência:

Uma sequência é uma função que associa a cada elemento n ∈ ℕ* um único elemento an ∈ ℝ.

f: ℕ* → ℝ


ℕ* = {1, 2, 3, 4, 5, … , n, … } à posição dos termos da sequência
f(1, 2, 3, 4, 5, … , n, … ) =
(a1, a2, a3, a4, a5, … , an, … )

1) Definição:


  • Chama-se Progressão Aritmética (PA) a sequência de números em que cada termo, a partir do segundo, é a soma do anterior com uma constante dada r.

2) Classificação


  • Crescente: cada termo é maior do que o anterior, ou seja, r > 0.
  • o Constantes: cada termo é igual ao anterior, então, r = 0.
  • o Decrescente: cada termo é menor que o anterior, logo, r < 0
