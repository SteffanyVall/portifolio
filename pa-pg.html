Reflexão:

Progressão Aritmética e Progressão Geométrica são conteúdos que são parecidos, por esse motivo foi um dos conteúdos fáceis de se aprender e fazer também. Os dois tem relação entre eles e funções afins, assim ficando até melhor de aprender e fazer.


Sequência:

Uma sequência é uma função que associa a cada elemento n ∈ ℕ* um único elemento an ∈ ℝ.

f: ℕ* → ℝ


ℕ* = {1, 2, 3, 4, 5, … , n, … } à posição dos termos da sequência
f(1, 2, 3, 4, 5, … , n, … ) =
(a1, a2, a3, a4, a5, … , an, … )



PROGRESSÃO ARITMÉTICA


1) Definição:


  • Chama-se Progressão Aritmética (PA) a sequência de números em que cada termo, a partir do segundo, é a soma do anterior com uma constante dada r.
  • Progressão Aritmética (PA) é a sequência de números na qual a diferença entre cada termo (a partir do segundo) e o termo anterior é constante e representado por r.

2) Classificação


  • Crescente: cada termo é maior do que o anterior, ou seja, r > 0.
  • o Constantes: cada termo é igual ao anterior, então, r = 0.
  • o Decrescente: cada termo é menor que o anterior, logo, r < 0

PROGRESSÃO GEOMÉTRICA


1) Definição:


  • Chama-se Progressão Geométrica (PG) a sequência de números em que cada termo, a partir do segundo, é o produto do anterior por uma constante dada q.
  • Progressão Geométrica (PG) é a sequência de números na qual o quociente entre cada um dos termos (a partir do segundo) e o seu termo anterior é constante e representado por q.


Exemplos de PGs:



2) Classificação


  • Crescente: cada termo é maior do que o anterior.
    • PG com termos positivos: q > 1.
    • PG com termos negativos: 0 < q < 1

  • Constantes: cada termo é igual ao anterior
    • PG com todos os termos nulos: a1 = 0 e q qualquer.
    • PG com termos iguais e não nulos: a ≠ 0 e q = 1.

  • Decrescente: cada termo é menor que o anterior.
    • PG com termos positivos: 0 < q < 1
    • PG com termos negativos: q > 1.

  • Alternantes: cada termo tem sinal contrário ao do termo anterior. Isso ocorre quando q < 0.

  • Estacionárias: Apenas o primeiro termo é diferente de zero: a1 ≠ 0 e q = 0.
